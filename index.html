Bienvenidos

Este es nuestro sitio del taller 1 de Desarrollo Web. Aquí practicamos HTML, CSS y Javascript básicos: estructura semántica, navegación multipágina, listas, tablas, multimedia, formularios e interacciones con el DOM. Usamos una fuente de Google Fonts, un menú superior fijo, y enlaces con estados de hover y activo. Evitamos caché para el cliente y permitimos indexación por robots.

El JavaScript se encarga de marcar en el menú a la página actual, aplicar scroll suave al ir a las demas paginas, y resaltar secciones cuando el mouse está encima. También añadimos validaciones de formularios en la página de contacto sin usar popups, mostrando mensajes de error al lado de cada campo si llega a ser el caso.

A nivel de tablas, aplicamos el esquema de filas alternadas y encabezado con fondo sólido; en cronología, las secciones tendrán borde y fondo en gradiente, además de viñetas y numeración con color distinto al texto.

El menú es responsive: en pantallas pequeñas se colapsa con un botón. No usamos frameworks para mantenernos en lo que pide la materia en esta etapa: HTML, CSS y JS puro. El código está organizado y comentado lo justo para que sea fácil de seguir.

La estructura de todas las páginas será consistente: mismo encabezado y pie de página, y un contenedor central con los contenidos específicos de cada sección (sobre nosotros, cronología, intereses, videos, links, imágenes y contacto).
